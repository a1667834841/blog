每年，每季，每月，每周的最后一天

当年第一天：

SELECT DATE_SUB(CURDATE(),INTERVAL dayofyear(now())-1 DAY);

当年最后一天：

SELECT concat(YEAR(now()),'-12-31');

当前week的第一天：

select date_sub(curdate(),INTERVAL WEEKDAY(curdate()) + 1 DAY);

select date_sub(curdate(),INTERVAL WEEKDAY(curdate()) - 5 DAY);
