如果mian包下有多个go文件，会出现下面的错误

```
command-line-arguments
```

该出错原因属于go的多文件加载问题，采用go run命令执行的时候，需要把待加载的.go文件都包含到参数里面。通过go run *.go(目录里面没有test.go才行)

或者配置IDE以package包模式就不会报错。
